[ 리액트 컴포넌트에서 데이터 전달하기 ]

-일반적으로 리액트에서 데이터는 하위 컴포넌트에서 props를 통하여 전달함
-일반적인 함수는 바깥쪽에 선언하면 전역 변수, 함수 내부에서 선언하면 지역 변수로 사용했으나, 컴포넌트는 자식 컴포넌트가 부모 컴포넌트의 변수나 함수 등을 직접 호출할 수 없음
why) 컴포넌트는 함수가 아니기 때문,,

1.부모 컴포넌트에서 자식 컴포넌트로 값 전달하기
-> Props Down : 속성을 통하여 값을 전달함
-형태 : <컴포넌트명 속성명=값 />
-

2.자식 컴포넌트에서 부모 컴포넌트로 값 전달하기 (함수 호출하는 방법)
-함수 호출할 때 값을 전달할 수 있음
-> Props Function Down & Up : 함수를 내려보내고 호출 받음
-프롭스 다운할 때 펑션을 호추할 수 있음 그 함수에서 값을 보내기로 되어 있으면 그 값이 보내짐
-서로 형제인 컴포넌트는 서로에게 직접 무언가를 할 수 없는데 이럴때도 이 방법이 유용함 (부모를 관제탑으로 쓰는 것임)
-형태 : <컴포넌트명 속성명={부모함수명} />
-이렇게 하면 자식 컴포넌트에서 속성으로 보낸 부모함수를 호출할 수 있음
-이때 부모함수가 값을 받게 하면 값도 전달됨
-변수명은 내가 지을 수 있음
⭐이거는 직계부모자식 관계에서 가능함 (부모의 부모의 부모라면... 계속 함수 연결해서 내려보내야함....아니면 다른 방법으로...!)


3.형제 컴포넌트는 서로 값을 어떻게 전달하나?
-> 자식에서 부모로 전달하고, 부모가 다른 자식으로 전달하면 됨
