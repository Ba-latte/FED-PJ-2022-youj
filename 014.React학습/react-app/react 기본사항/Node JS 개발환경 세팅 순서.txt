[ Node JS 개발환경 세팅 순서 ]
-뷰js, 리액트 공통
1.깃허브 소스 클론하기
2.노드 cli 환경 설치하기
(1) 기존 프로젝트 폴더 이름 바꾸기
ex) aaa -> aaa2
->탐색기에서 직접 변경해주기 (vscode닫고 해야 함~~)

(2) vscode 코드 열기

(3)깃허브에 올린 이름과 같은 프로젝트 생성

(4)프로젝트에 들어가서 확장 의존성 프로그램 설치

(5)vscode닫고 탐색기에서 이름바꾼 폴더의 개발소스만 지우고 복사,붙여넣기

(6)이름바꾼 백업파일 지우기

(7)다시 cli 환경에서 npm run dev로 로컬서버 실행시켜서 동일하게 작동됨을 확인하기
