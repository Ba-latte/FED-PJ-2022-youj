[2]메뉴 버튼 모듈
    -모듈명 : MenuBtn (JS는 첫글자 대문자/CSS는 전부 소문자)
    (1) 구조정의 :
    -Root> section >img Box + title Box + button Box  
    -섹션으로 전체를 아우르는 박스를 하나 더 만들고 그 밑의 요소들을 반복시킴

    (2) 객체화 :
    {
        "isrc": "이미지 경로",
        "tit": "작은 타이틀^큰 타이틀", 👈 만약 작은 타이틀이 없다면 "^큰 타이틀" 이라고만 써서 구분자로 구분해주는 형식을 쓰면 됨
        "btn": "버튼 문구",
        "link": "이동카테고리"
    }
    ->"tit"속성은 ^로 데이터를 구분함
    ->컴포넌트별 객체화 데이터 반영 (너무 기본적인 기능 정의라서 위로 뺌 ㅎㅎ)

    (3) 기능정의 :
    1) 버튼에 오버시 하이라이트
        -mouseover/mouseout 이벤트 + CSS
    2) 버튼 클릭시 페이지 이동
        -(click) + location(라우트 이동)


[3]비디오 소개 모듈

    (1) 구조정의 :
    -Root > video box(>video+img 태그) + title box(>heading + p 태그)

    (2) 객체화 :
    {
        "vsrc": "유튜브 동영상 경로",
        "stit": "작은 타이틀",
        "btit": "큰 타이틀",
        "sum": "요약 소개",
        "desc": "설명",
        "link": "관련 링크",
    }

    (3) 기능정의 :
    1) 동영상 영역 클릭시 동영상 재생/멈춤
    2) 관련 링크 새창열기
    3) 


[4]비디오 스와이프 모듈
    -모듈명 : VidSwipe (JS / CSS / data)
    (1) 구조정의 :
    -스와이퍼 모듈 + 비디오 재생 모듈

    (2) 객체화 :
    {
        isrc: "",
        vsrc: "",
        cat: "",
        tit: "",
    }
    -isrc : 이미지 경로
    -vsrc : 유튜브 URL
    -cat : 카테고리
    -tit : 타이틀
    
    (3) 기능정의 :
    1) 비디오 메뉴 스와이프 기능(스와이퍼)
    2) 비디오 메뉴 클릭시 별참 동영상 재생 기능(닫기 버튼)
