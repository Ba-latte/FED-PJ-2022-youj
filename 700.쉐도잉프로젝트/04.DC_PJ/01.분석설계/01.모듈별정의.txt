[2]메뉴 버튼 모듈
    -모듈명 : MenuBtn (JS는 첫글자 대문자/CSS는 전부 소문자)
    (1) 구조정의 :
    -Root> section >img Box + title Box + button Box  
    -섹션으로 전체를 아우르는 박스를 하나 더 만들고 그 밑의 요소들을 반복시킴

    (2) 객체화 :
    {
        "isrc": "이미지 경로",
        "tit": "작은 타이틀^큰 타이틀", 👈 만약 작은 타이틀이 없다면 "^큰 타이틀" 이라고만 써서 구분자로 구분해주는 형식을 쓰면 됨
        "btn": "버튼 문구",
        "link": "이동카테고리"
    }
    ->"tit"속성은 ^로 데이터를 구분함
    ->컴포넌트별 객체화 데이터 반영 (너무 기본적인 기능 정의라서 위로 뺌 ㅎㅎ)

    (3) 기능정의 :
    1) 버튼에 오버시 하이라이트
        -mouseover/mouseout 이벤트 + CSS
    2) 버튼 클릭시 페이지 이동
        -(click) + location(라우트 이동)

[4]비디오 스와이프 모듈
    -모듈명 : VidSwipe (JS / CSS / data)
    (1) 구조정의 :
    -스와이퍼 모듈 + 비디오 재생 모듈

    (2) 객체화 :
    {
        isrc: "",
        vsrc: "",
        cat: "",
        tit: "",
    }
    -isrc : 이미지 경로
    -vsrc : 유튜브 URL
    -cat : 카테고리
    -tit : 타이틀
    
    (3) 기능정의 :
    1) 비디오 메뉴 스와이프 기능(스와이퍼)
    2) 비디오 메뉴 클릭시 별참 동영상 재생 기능(닫기 버튼)


[5]캐릭터 모듈
    -모듈명 : Cat (JS / CSS / data)
    (1) 구조정의 :
    -제목 + 스와이퍼 모듈
    ->하위 스와이퍼 모듈(이미지+이름)

    (2) 객체화 :
    {
        idx: "",
        tmsrc: "",
        cname: "",
        cdesc: "",
        facts: "",
        alignment: "",
        type: "",
    }
    -idx : 인덱스번호 (유일키==기본키)
    -tmsrc : 썸네일 이미지 경로
    -cname : 캐릭터 이름
    -cdesc : 캐릭터 설명
    -facts : 캐릭터 명세
    -alignment : 캐릭터 분류(역할 히어로/빌런/복잡)
    -type : 캐릭터 타입(개인/단체)
    
    (3) 기능정의 :
    1) 캐릭터 메뉴 스와이프 기능(스와이퍼)
    2) 캐릭터 메뉴 클릭시 캐릭터 상세 페이지 이동(라우터 사용)



    [5]상세페이지 모듈 : 페이지 구성 모듈(모듈 폴더에 넣는 게 아니라 페이지 모듈이라고 생각하면 됨!)
    -모듈명 : Detail (JS / CSS / data)
    (1) 구조정의 :
    -배너 모듈 + 캐릭터명 + 캐릭터소개 + 캐릭터 명세 + 연관 캐릭터 모듈

    (2) 객체화 : 
    1) 캐릭터 모듈 data 사용
    2) 배너 모듈 data 사용
    
    (3) 기능정의 :
    1) 캐릭터 메뉴에서 넘어온 캐릭터 키를 가지고 데이터 값을 세팅하기
