[ 로그인 프로세스 ]

1.회원 정보를 조회하여 아이디 비번 일치시 로그인 정보를 서버 세션에 기록함
-세션이 출발된다고 하여 세션 스타트라고함
👉우리는 서버 세션 대신 로컬스토리지를 사용해서 로컬스토리지 변수에 기록함

2.서버 세션에 로그인 정보가 있을 경우 사용자 페이지에 로그인 상태 표시를 해주기
👉우리는 로컬스토리지 변수를 조회해서 현재 로그인한 회원의 로그인 상태를 화면에 표시함

3.로그인 관련 기능 업데이트하기
(1)회원명으로 환영 메시지 띄우기
(2)회원가입, 로그인 버튼 사라지기
(3)로그 아웃 버튼 나타나기
(4)회원 관련 권한에 따른 조작 버튼 등 보이기 (추가적인 기능임)
-ex) 관리자 페이지 버튼, 게시판 글쓰기/지우기 버튼 등
-회원이면 글쓰기, 자기 글 지우기 등 가능, 비회원은 볼 수만 있음

4.로그아웃 버튼 클릭시 서버의 로그인 세션을 종료하기 (소멸됨)
👉우리는 로그인 로컬스토리지 변수를 삭제함!
